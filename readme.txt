PDF 壓縮大師 (PDF Master) - Client-side PDF Compressor

這是一個基於瀏覽器執行的純前端 PDF 壓縮工具，強調隱私保護（100% 本地運算）與高效能。

功能特色

100% 本地執行：使用 WebAssembly 技術，檔案不上傳伺服器。

多種壓縮模式：支援結構優化 (無損) 與 數位重掃描 (強力壓縮)。

多語言支援：內建 13 國語言介面 (i18n)。

離線可用：無需網路連線即可處理文件。

技術棧

React + Vite

Tailwind CSS

PDF.js / jsPDF / pdf-lib (處理核心)

Lucide React (圖標庫)

如何開始

安裝依賴：

npm install


啟動開發伺服器：

npm run dev


建置生產版本：

npm run build


部署

本專案可直接部署於 Vercel 或 Netlify。
